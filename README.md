# 輕贏實驗室 (Qingying Lab)

> **輕贏** = 輕盈的狀態 + 比昨天的自己更進步就是贏
> 
> 不是跟別人比較，不是追逐社會標準。
> 
> 是把自己活好，然後陪別人也活好。

我是宇宙媽媽賞。我要成為帶自己「走出趣、翻身、找到伯樂」的那個人。

這個實驗室，記錄我把自己當實驗品的一切過程。

## 核心定義

- **走出趣**：先幫自己走出去，走出去之後就能幫助別人。
- **翻身**：從迷惘到有方向的過程。
- **伯樂**：先當自己的伯樂，然後成為別人的伯樂。
