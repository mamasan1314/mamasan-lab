# 輕贏實驗室的信念與原則

## 核心定義

| 詞 | 定義 |
|---|---|
| **輕贏** | 輕盈的狀態 + 比昨天的自己更進步就是贏（不是跟別人比） |
| **走出趣** | 先幫自己走出去，走出去之後就能幫助別人 |
| **翻身** | 從迷惘到有方向的過程 |
| **伯樂** | 先當自己的伯樂，然後成為別人的伯樂 |

## 實驗室原則

- **簡單**：不再把人生過得太複雜。
- **自然**：越認識自己，越不用演。
- **勇敢**：認識自己需要勇敢。

## 三心三自

### 三心
- 真心
- 開心
- 玩心

### 三自
- 自信（信任自己）
- 自在（與自己在一起）
- 自由（拿掉限制性信念）
